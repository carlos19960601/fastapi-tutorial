
**创建虚拟环境**

```
python3 -m venv env/
```

**激活虚拟环境**

```
source env/bin/activate
```

**退出虚拟环境**

```
deactivate
```

**安装 FastAPI 和 uvicorn**

```
pip install fastapi uvicorn
```

**运行服务器**

```
uvicorn main:app --port 8000 --reload 
```

# 安装依赖

```
pip install -r requirements.txt
```

# 运行

```
python main.py
```
